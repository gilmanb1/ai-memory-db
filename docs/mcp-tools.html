MCP Tools

Model Context Protocol tools for AI-powered coding agents

The memory system includes an MCP server that gives any MCP-compatible AI tool on-demand read/write access to the knowledge base. It communicates via stdio using JSON-RPC 2.0 — no separate process to manage. The installer registers it automatically in your settings.json.

memory_search
Search all memory types by semantic query. Returns ranked results from facts, guardrails, procedures, error solutions, and decisions.
Parameters
NameTypeRequiredDescription
querystringyesThe search query text
typesstring[]noFilter by type: facts, guardrails, procedures, error_solutions, decisions. Default: all.
limitintegernoMax results per type (default 10)
Example
memory_search(query="retry logic", types=["facts", "guardrails"], limit=5)
memory_store
Store a fact, decision, guardrail, procedure, or error solution in the knowledge base.
Parameters
NameTypeRequiredDescription
textstringyesThe content to store
typestringyesOne of: fact, decision, guardrail, procedure, error_solution
importanceintegernoImportance score 1-10 (default 7)
file_pathsstring[]noFile paths to associate with this memory
Example
memory_store(text="API rate limit is 1000 req/min", type="fact", importance=8, file_paths=["src/middleware.py"])
memory_guardrail
Create a guardrail — a protective rule that surfaces with highest priority when working with specific files.
Parameters
NameTypeRequiredDescription
warningstringyesThe guardrail warning text
rationalestringnoWhy this guardrail exists
consequencestringnoWhat happens if the guardrail is violated
file_pathsstring[]noFile paths this guardrail protects
Example
memory_guardrail(warning="Never modify auth.py without running security tests", rationale="PCI compliance", consequence="Audit failure", file_paths=["src/auth.py"])
memory_check_file
Get all guardrails, procedures, facts, and error solutions associated with a file path.
Parameters
NameTypeRequiredDescription
file_pathstringyesThe file path to check
Example
memory_check_file(file_path="src/auth.py")
